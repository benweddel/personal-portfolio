API App

This Marvel themed app pits superheroes against each other based on their popularity. Offering a wide range of characters to choose from, feel free to experiment beyond the drop-down options. I added in a fun animation sequence to enact the “Popularity Battle”.

Tools & Tech Highlights:

  • AJAX & JSON
  • RESTful API
  • JavaScript & jQuery
  • Auto-complete drop-down menu
