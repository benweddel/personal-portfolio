We have a body-wrapper which is acting like a dummy body and contains our sections. This template is one page so the whole content should be in one html file which is index.htmlstrong>. In our case each section is a separate page and we have five of them. Home, About, Skills, Portfolio and Contact.

The important attribute in here is ID of each section because we use it as an identifier so it must be something clear.

Pages

Each page needs two inner container/wrapper to works. inner-wrapper gives us a scrolling ability and contain some proper paddings.

So from here when we talk about for example home page contents we are referring the contents between contain of that section.
