Colors

choose diffeent color or add a new one

Change the default color

We have 8 pre defined style as you can see them in demo page. to change the default color all you need to edit index.html file at line 22:

Each one of styles has a css file located in "css/styles" just change the "default.css" with any of them like "red.css"

Notice:We need to edit circular charts color from javascript. open file custom.js and at line 210 you should see this:

//Circular Easy pie chart
  demandPage.find('.chart').easyPieChart({
      scaleColor: false,
      barColor: '#D0482B',
      lineWidth: 10,
      trackColor: '#4a504f',
      lineCap: 'butt',
      animate: 1000,
      size: 70,
      onStep: function(from, to, percent) {
          $(this.el).find('span').text(Math.round(percent) + '%');
      }
  });

Supppose we want to change to red.css we need to open red.css file and pick the color which is #d02b2b in this case and enter it in barColor



Add new color

You can have your own colors these 8 predefined are just some examples. to add a new one open "default.css" change all occurence of "#D0482B" with your own color and save the file with a different name like "mycolor.css" in "css/styles" folder. Then like the first case just add "mycolor.css" to "index.html" file

Notice:All colored elements have the same color beside Vertical bar chart which has a gradient so we have two colors here. If you need to add a new color be sure to also change the second color. you can find it in color stylesheets at line 56: 
  .vertical-bar li .bar{
    background: #d0482b; 
    background: -moz-linear-gradient(top,  #d0482b 1%, #9b2020 100%); 
    background: -webkit-gradient(linear, left top, left bottom, color-stop(1%,#d0482b), color-stop(100%,#9b2020)); 
    background: -webkit-linear-gradient(top,  #d0482b 1%,#9b2020 100%);
    background: -o-linear-gradient(top,  #d0482b 1%,#9b2020 100%);
    background: -ms-linear-gradient(top,  #d0482b 1%,#9b2020 100%);
    background: linear-gradient(to bottom,  #d0482b 1%,#9b2020 100%);
    filter: progid:DXImageTransform.Microsoft.gradient( startColorstr='#d0482b', endColorstr='#9b2020',GradientType=0 );

}

Just change the second color which is #9b2020 in this case
